#  Finance Loan Data Analysis Project
This project focuses on analyzing loan data to understand lending performance and borrower behavior


## Overview
This project analyzes loan data to evaluate lending performance and identify key financial trends. The goal is to understand how loans are distributed, how borrowers behave, and what factors contribute to good or bad loans.



##  Objectives
- Track total loan applications over time
- Analyze total funded and received amounts
- Evaluate borrower financial health using debt-to-income ratio
- Study average interest rates across loans
- Compare good loans vs bad loans
- Identify patterns in lending based on region, employment, and loan purpose



## Key Metrics (KPIs)
- Total Loan Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average Debt-to-Income Ratio (DTI)  



## Analysis Performed

### 1. Loan Quality Analysis
- Good Loan vs Bad Loan comparison
- Application percentages
- Funded vs received amounts

### 2. Trend Analysis
- Monthly loan trends (seasonality & growth)
- Lending patterns over time

### 3. Regional Analysis
- Loan distribution across different regions
- Identification of high and low performing areas

### 4. Loan Term Distribution
- Analysis of loan durations
- Understanding borrower preferences

### 5. Employment Analysis
- Impact of employment length on loan applications and repayment

### 6. Loan Purpose Breakdown
- Why borrowers take loans (e.g., personal, business, etc.)

### 7. Home Ownership Impact
- Relationship between home ownership and loan behavior



##  Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib / Seaborn / Plotly)
- Jupyter Notebook



## Insights
- Clear distinction between good and bad loans based on repayment behavior
- Certain regions show higher lending activity
- Employment length influences loan approval and repayment
- Loan purpose plays a key role in risk assessment



## Conclusion
This project demonstrates how data analysis can be used to evaluate financial performance and support better lending decisions. It highlights important patterns that can help reduce risk and improve loan management strategies.
